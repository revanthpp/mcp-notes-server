# AI Systems

An AI system is more than a model. It includes the application, prompts, tools,
data, policies, people, and operational controls around the model.

Useful questions:

- Which component owns user consent?
- Which tools can cause side effects?
- Where does retrieved data go?
- What evidence will an auditor have later?

Treat model output as an input to the rest of the system, not as trusted code.
